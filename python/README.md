## Tầm nhìn

Chương trình **CocAgent Python Learning Path** được xây dựng trên một nguyên tắc cốt lõi:

> *Python dễ bắt đầu nhưng khó thành thạo. Chương trình này lấp đầy khoảng cách đó — từ người mới hoàn toàn đến kỹ sư Python thực thụ, qua hệ sinh thái 5 repo có cấu trúc rõ ràng và kết nối chặt chẽ.*

Không phải mọi người đều cần đi toàn bộ 5 repo. Mỗi nhánh chuyên ngành (DSA, Data, Automation) là một hướng đi độc lập — nhưng tất cả đều được xây trên cùng một nền móng vững chắc.

### Journey → Mastery: Nâng cấp từng khái niệm

| Journey dạy | Mastery nâng lên |

| Journey cung cấp | DSA cần để |
|---|---|
| List, Dict, Set | Understand dynamic array, hash table |
| Function & recursion | DFS, BFS, Backtracking, DP |
| OOP cơ bản | Tree node, Graph vertex class |
| `sort()`, `sorted()` | Understand sorting algorithms |

### Mastery → Data / Automation: Tools & Patterns

| Mastery cung cấp | Data Science dùng | Automation dùng |
|---|---|---|
| Generator | Lazy data loading | Stream processing |
| Decorator | Pipeline decorator | Task decorator |
| Async/await | Async HTTP calls | Concurrent scraping |
| Type hints | pandas type safety | CLI type validation |
| Testing (pytest) | Test data pipelines | Test automation scripts |

## Triết lý giảng dạy

**1. Python dễ đọc không có nghĩa là dễ viết đúng.** Mỗi tính năng được học kèm với *trade-off* và *khi nào không nên dùng*.

**2. Từ script đến system.** Bắt đầu với script 10 dòng, kết thúc với hệ thống có test, packaging, CI/CD.

**3. Test không phải tùy chọn.** Pytest được giới thiệu từ Mastery W04 và bắt buộc từ DSA/Data/Automation.

**4. Dự án thực > tutorial lý thuyết.** Mỗi khóa có ít nhất 3 dự án hoàn chỉnh mà bạn có thể cho vào portfolio.

**5. Pythonic code là mục tiêu, không phải kết quả ngẫu nhiên.** Mỗi solution đều được review theo tiêu chí Pythonic — không chỉ "chạy được".
